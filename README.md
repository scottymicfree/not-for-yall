# Alpha Delta Vualt

A local workspace builder with a one-click Windows launcher.

## What it does
- Mounts a local workspace folder
- Creates an easy output path: `AlphaDeltaVualt_Downloads` inside the mounted workspace
- Lets you chat with the local builder assistant
- Can scaffold a basic FiveM resource locally

## Start
Double-click `START_LUCY.bat` on Windows, or run `npm install` then `npm run dev`.
